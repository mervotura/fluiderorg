Teknologi

Berikut adalah teknologi yang digunakan dan dikembangkan untuk mengisi stack dari program fluider. Bilamana di dalam kuriklum adalah high level material, area ini membahas teknologi yang dikembangkan, baik yang telah maupun yang akan menjadi material kurikulum berikutnya. Khususnya untuk area pustaka akan berkembang searah dengan aktifitas komunitasnya, khususnya perpustakaan Javascript untuk mencakup semua yang mengisi dan menguatkan kurikulum fluider.

Responsive

Responsive Web Design adalah sebuah teknik yang digunakan untuk membuat layout website menyesuaikan diri dengan tampilan device pengunjung web browser, baik ukuran maupun orientasinya. Untuk mendesain web Responsive, Fluider menggunakan 2 framework CSS, yaitu : Twitter Bootstrap, Zurb Foundation.

Programming

Untuk Pemula yang ingin bergelut dalam dunia desain, diharuskan mengetahui beberapa hal dasar dalam mendesain web dan dasar dalam mendesain sebuah web di bagi menjadi 3 bahasa pemrogaman, bahasa pemrogaman tersebut adalah : HTML, CSS, JavaScript.
